下载：https://github.com/cornradio/pausemygameW/releases/ 

1. 双击启动exe
2. 点击 Help-pssuspend 下载 pssuspend.exe 并放到exe同目录下
3. 第一次使用需要手动点开 pssuspend.exe ，同意使用许可（微软要求）
4. 然后添加游戏exe名称，并进行游戏的pause、resume等操作。
